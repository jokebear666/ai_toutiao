  • [arXiv260105] The permutation group of Reed-Solomon codes over arbitrary points

    • tags: [other], [coding theory], [Reed-Solomon codes, permutation group, automorphism group, affine permutations, evaluation points]

    • authors: Eduardo Camps-Moreno, Jun Bo Lau, Hiram H. López, Welington Santos

    • institution: Virginia Tech, KU Leuven, University of Wisconsin-Stout

    • link: https://arxiv.org/pdf/2601.00122

    • contributions: 1. Proves that the permutation group of a Reed-Solomon code is precisely the set of degree-one polynomials (affine permutations) that leave its set of evaluation points invariant. 2. Provides a straightforward, elementary algebraic proof for this characterization, applicable to arbitrary evaluation sets. 3. Generalizes and simplifies the proof for well-known special cases where the evaluation set is the entire finite field or the multiplicative group.

    • thumbnail: https://pub-9ba4a5dae3bf4fc7b7d26411a74e92db.r2.dev/thumbnails/21b259b314c4815c0a20b42d7c453b8d4e754d4b81e7ac1ffe62ccf4ab62c34c_w640_q70.webp

    • Simple LLM Summary: This paper studies the permutation group (automorphism group) of Reed-Solomon codes. It proves that this group consists exactly of the affine permutations (polynomials of degree one) that map the code's evaluation set to itself. This result provides a simple, unified proof for previously known special cases and clarifies the structure of the code's symmetries for arbitrary evaluation points.
